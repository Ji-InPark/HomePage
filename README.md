# jin react starter

## 사용법(IntelliJ 기준)

### 프로젝트 이름 변경
1. Repository clone
2. 프로젝트 폴더 이름 변경
3. File -> Project Structure
4. Project -> 이름 변경

### 프로젝트 모듈 설정
1. File -> Project Structure
2. Module -> + -> Import Module
3. 현재 프로젝트 루트 폴더를 선택 -> 적용

### 프로젝트 의존성 모듈 설치
```shell
yarn install
```
